# 🎂 Happy Birthday Web App

A birthday surprise built with vanilla JS, Node and Express:

- 🎉 **Start menu** with confetti, floating balloons and a shimmering title
- 💌 **A love letter** typed out word-by-word like it's being written for her
- 🎮 **Noughts & crosses** — she plays hearts ♥ against Cupid 🏹 (slightly
  beatable on purpose), or switch to 2-player mode and play together

## Run it

```bash
npm install
npm start
```

Then open <http://localhost:3000>.

## Make it hers

Everything personal lives in **`public/js/config.js`**:

- `name` — her name on the start menu
- `message` — the paragraphs of the letter
- `signature` — your sign-off

No other files need editing.
